# Border & Outline Panel

<hr/>

Border & Outline Panel은 Css가 표현될 Dom의 border와 outline 스타일을 편집을 위한 UI의 모음입니다.

Main Container를 기반으로 Before, After Container를 모두 편집할 수 있습니다.

<hr/>

## Detail UI
  - outline
  - border(basic)
- border radius
- border width
- border color
- border style
- border(gradient)
- border image outset
- border image slice
- border image repeat
- border gradient step editor
