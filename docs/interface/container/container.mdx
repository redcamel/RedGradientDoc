# Container Panel

<hr/>

Container Panel은 ```Css가 표현될 Dom``` 스타일을 편집을 위한 UI의 모음입니다.


:::info
패널내 모든 에디트 UI는 현재 활성화된 Viewpor window내의 타겟 컨테이너를 편집합니다.
:::

:::info
단축키 : ```F1``` 키를 통해 ```on/off``` 할 수 있습니다.
:::
<hr/>
