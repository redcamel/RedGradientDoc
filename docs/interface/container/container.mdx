# Container Panel

<hr/>

Container Panel은 Css가 표현될 Dom 스타일을 편집을 위한 UI의 모음입니다.

Main Container를 기반으로 Before, After Container를 모두 편집할 수 있습니다.

단축키 : ```F1``` 키를 통해 ```on/off``` 할 수 있습니다.
<hr/>

## Detail UI
  - size
  - position
  - background color
  - box sizing
- box shadow
