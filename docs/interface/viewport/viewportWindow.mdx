## Transform Mode UI
![ui_viewport_window_transform_mode](./img/ui_viewport_window_transform_mode.png)
### Container Mode
```text
컨테이너 사이즈를 마우스 + 키보드를 통해 수정 할 수 있습니다.
```
- 단축키 : ```CTR + 1```
- move
- resize

![ui_transform_container](./img/ui_transform_container.png)

### Gradient Mode
```text
그라디언트 사이즈를 마우스 + 키보드를 통해 수정 할 수 있습니다.
```
- 단축키 : ```CTR + 2```
- move
- resize

![ui_transform_gradient](./img/ui_transform_gradient.png)

### Radius Mode
```text
컨테이더의 border-radius를 마우스를 통해 수정 할 수 있습니다.
```
- 단축키 : ```CTR + 3```

![ui_transform_radius](./img/ui_transform_radius.png)

### None Mode
```text
Transform Editor 없이 현재 상태를 확인하고 싶을때 None 모드를 사용합니다.
```
- 단축키 : ```CTR + 4```
