### Project name edit
![](./img/ui/ui_project_name.png)
- ```프로젝트 이름```을 수정 할 수 있습니다.
- 프로젝트를 저장할때 ```기본 이름```으로 지정됩니다.

<hr/>

### Project save/load
![](./img/ui/ui_save_load.png)
- #### 프로젝트를 생성/저장/로드 할 수 있습니다. ####

:::info
- 생성 : ```ALT + N```
- 저장 : ```CTR + S```
- 로드 : ```CTR + O```
:::



- #### undo/redo ####

:::info
- undo : ```CTR + Z```
- redo : ```CTR + SHIFT +Z```
:::
