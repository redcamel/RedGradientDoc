### project save/load
![](./img/ui/ui_save_load.png)
- #### 프로젝트를 생성/저장/로드 할 수 있습니다. ####

:::info
- 생성 : ```ALT + N```
- 저장 : ```CTR + S```
- 로드 : ```CTR + O```
:::



- #### undo/redo ####

:::info
- undo : ```CTR + Z```
- redo : ```CTR + SHIFT +Z```
:::
