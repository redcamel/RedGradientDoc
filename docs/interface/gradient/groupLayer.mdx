# Layer Panel

<hr/>

Layer Panel은 해당 컨테이너에 속해있는 ```group```과 ```layer```의 ```생성```, ```삭제```, ```이동```, ```복제```에 대한 기능을 제공합니다.

:::info
단축키 : ```F2``` 키를 통해 ```on/off``` 할 수 있습니다.
:::
