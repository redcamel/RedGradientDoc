Menu Bar

최상단 글로벌 Top Menu

  • project name edit
  • project save/load
  • project publish

project name edit​

  • 프로젝트 이름을 수정 할 수 있습니다.
  • 프로젝트를 저장할때 기본 이름으로 지정됩니다.

project save/load​

  • 프로젝트를 생성/저장/로드 할 수 있습니다.​

info
  • 생성 : ALT + N
  • 저장 : CTR + S
  • 로드 : CTR + O

  • undo/redo​

info
  • undo : CTR + Z
  • redo : CTR + SHIFT +Z

project publish​

  • 프로젝트 결과를 HTML로 다운받거나 Css 결과를 복사 할 수 있습니다.
